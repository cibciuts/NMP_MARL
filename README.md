This is the code for the paper "NEURAL MESSAGE-PASSING FOR LARGE-SCALE MULTI-AGENT REINFORCEMENT LEARNING", by Dr Kha Vo and Distinguished Professor Chin-Teng Lin.

# Dependencies
```
torch 1.1.0
schnetpack 
ase 3.17.0
torch-scatter 1.4.0
torch_geometric 1.3.2
torch-sparse 0.4.3
torch_cluster 1.4.5
```
